# Design & Brand Direction

## Product Goal

Beauty salon called Clair de Lune with polished and premium visual.

## Design System Defaults

- Fonts: Albert Sans by default

## Brand Direction

- Palette:
  - primary: `#c8a451`
  - base: `#ffffff`
  - common text: `#000000`
  - optional support: `#f2d7d0`, `#a3b8c8`
- Tone: premium, calm, clean, uncluttered
- Voice: use feminine references for the brand/customer context when writing UX copy
