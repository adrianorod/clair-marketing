## Conceito Visual

A lua como metáfora de ciclos, renovação e beleza natural.

Não é apenas um salão — é um espaço de transformação.
A identidade visual deve transmitir:

- **Sofisticação acessível** — não elitista, mas refinada
- **Acolhimento feminino** — quente, seguro, empoderador
- **Ouro** — valorização, cuidado premium sem ostentação
- **Voz** - use sempre referencias femininas ao falar da marca ou das clientes

## Paleta

### Primárias

```
Branco: #ffffff
→ Fundo principal claro (branco, bege ou cinza)
→ Transmite: limpeza, sofisticação, calma
→ Uso: backgrounds, overlays
```

```
Dourado Suave: #C9A96E
→ Acento principal
→ Transmite: luxo acessível, calor, feminilidade madura
→ Uso: destaques, logos, CTAs secundários
```

### Secundárias

```
Rosa Antigo: #D4A5A5
→ Transmite: feminino, delicado, não infantil
→ Uso: elementos decorativos, destaques leves
```

```
Cream: #FAF6F0
→ Transmite: limpeza, cuidado, orgânico
→ Uso: fundos claros, temas "dia"
```

### Evitar

- ❌ Rosa shocking / bubblegum
- ❌ Roxo vibrante
- ❌ Azul elétrico
- ❌ Verde neon

## Tom Visual

### Referências visuais

- Lua crescente como ícone identitário
- Gradientes sutis (não vibrantes)
- Decorações geométricas simples (círculos, linhas)
- Grade de pontos suave como textura de fundo
- Arcos e curvas como elementos de layout

---

## Hierarquia em Artes de Marketing

```
1. HEADLINE — Albert Sans, grande, impacto imediato
   → Máximo 8 palavras
   → Preferencialmente com verbo de ação ou emoção

2. VISUAL — preço, imagem, decoração
   → Que "para o olho" depois da headline

3. CTA — pill button, texto em caixa alta
   → Direto: "Assine agora", "Agende", "Venha conhecer"

4. MARCA — logo discreta mas presente
```

## Restrições Estéticas

- Nunca usar fontes diferenes das citadas explicitamente no manual
- Nunca usar gradientes de arco-íris
- Nunca usar imagens de banco de imagens genéricas sem curadoria
- Nunca usar texto em caps lock em headlines (apenas labels)
- Nunca usar mais de 3 cores por arte (+ neutros)
- Nunca colocar texto sem contraste adequado sobre imagem sem overlay
