# Copywriting — Clair de Lune

## Tom de Voz

- **Empoderador:** "Você merece"
- **Direto:** sem rodeios, sem texto corporativo
- **Caloroso:** próximo, não frio
- **Confiante:** afirma, não questiona

## Fórmulas de Headline

### Clubes de assinatura

```
[Benefício claro] + [frequência/facilidade]
→ "Cabelo perfeito toda semana."
→ "Unhas sempre impecáveis."
→ "Sua transformação começa aqui."
```

### Sazonais

```
[Emoção] + [destinatário]
→ "Para a mulher que merece tudo."
→ "Presentes que ela vai adorar."
```

### Promoções

```
[Urgência suave] + [benefício]
→ "Só essa semana: cuide-se mais."
→ "Aproveite antes de acabar."
```

## CTAs por contexto

| Contexto | CTA |
|----------|-----|
| Clube | Assine agora |
| Agendamento | Agende agora |
| Story | Arraste e assine / Venha conhecer |
| Presente | Presenteie agora |
| Promoção | Aproveite agora |

## Regras

- Headline: máximo 8 palavras
- Ponto final em headlines afirmativas
- CTA: imperativo, nunca "Clique aqui"
- Badge: 1-2 palavras, tudo maiúsculo
- Preço: sempre com "R$", nunca "reais"
