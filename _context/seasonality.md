# Calendário Sazonal — Clair de Lune

## Classificação por Mês

| Mês | Temporada | Eventos-Chave | Estratégia |
|---|---|---|---|
| Janeiro | Alta | Verão, Carnaval | Combos de escova + hidratação para o verão. Foco em Venda |
| Fevereiro | Alta | Carnaval | Preparação para festas. Makes e penteados duradouros. Venda |
| Março | Alta | Dia da Mulher (8/3) | Autoestima + promoção especial. Mix Venda + Autoestima |
| Abril | Normal | Páscoa | "Presentear-se" — combos gift. Venda leve |
| Maio | Alta | Dia das Mães (2º domingo) | Principal data do semestre. Presente para mãe, clubes |
| Junho | Normal | Noivas, Festas Juninas | Pacotes para noivas, looks festa. Venda direcionada |
| Julho | Baixa | Férias escolares | Dicas de manutenção em casa. Reforçar conteúdo |
| Agosto | Baixa | — | Reforço de marca, bastidores, storytelling equipe |
| Setembro | Baixa | — | Educação capilar profunda, tutoriais, dicas técnicas |
| Outubro | Normal | — | Preparação para Q4. Aquecimento Black Friday |
| Novembro | Alta | Black Friday | Mega ofertas anuais. Venda direta máxima |
| Dezembro | Alta | Natal, Ano Novo | Looks para festas, vale-presente. Venda + Autoestima |
